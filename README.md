# cloudcore-cms.github.io

Documentation and marketing site for [Cloudcore CMS](https://github.com/cloudcore-cms/cloudcore-cms).

Built with [Astro](https://astro.build). Deployed automatically to GitHub Pages on push to `main`.

## Development

```bash
npm install
npm run dev
# http://localhost:4321
```

## Build

```bash
npm run build
# Output in dist/
```
